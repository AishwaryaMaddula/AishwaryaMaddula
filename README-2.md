# Aishwarya Maddula — Data Portfolio

This is a simple static portfolio (light theme) designed for GitHub Pages.

## Local preview
Open `index.html` in your browser.

## Deploy to GitHub Pages
1. Create a new public repo named `AishwaryaMaddula.github.io` (or any name).
2. Upload these files (or push via git).
3. In the repo, go to **Settings → Pages** and:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (root)
4. Your site will be live at `https://AishwaryaMaddula.github.io/`.

## Customize
- Edit text in `index.html` and `projects.html`.
- Update links to your repositories in the **Projects** page.
- Tweak look & feel in `style.css`.
